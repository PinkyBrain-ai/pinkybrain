## 🤝 Sponsoring

Les entreprises ou contributeurs qui souhaitent sponsoriser PinkyBrain reçoivent en échange :
- Mention dans le README
- Logo dans les releases
- Visibilité sur le repo GitHub

**Ce que le sponsor n'achète PAS :** du contrôle, des features exclusives, ou de l'influence sur la direction du projet. Le projet reste libre.
