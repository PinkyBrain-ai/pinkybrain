## ✨ ما هو PinkyBrain؟

يقوم PinkyBrain بربط الأجهزة في شبكة ذكاء اصطناعي نظير-لنظير. أجهزتك تتواصل فيما بينها، وتشارك المعالجة والنماذج — بدون اعتماد على السحابة، بدون خادم مركزي، بدون نقطة فشل واحدة.

**الإصدار v5 يضيف الشبكة العامة:** انضم إلى شبكة عالمية من المعالجة المركزية والذاكرة ووحدات معالجة الرسوميات ونماذج الذكاء الاصطناعي المشتركة. شبكتك الخاصة تبقى خاصة. الشبكة العامة هي طبقة إضافية تختار الانضمام إليها.

**باختصار:** مثل BitTorrent، لكن للذكاء الاصطناعي. تشارك المعالجة، وتصل إلى أكثر من 50 نموذج. بياناتك تبقى على جهازك. مشفرة من الطرف إلى الطرف. دائماً.

## 🚀 البدء السريع

### المتطلبات المسبقة
- Python 3.12+
- [Ollama](https://ollama.ai) يعمل محلياً (أو نقطة نهاية نموذج سحابي)
- (اختياري) [Tailscale](https://tailscale.com) لاكتشاف تلقائي للأقران

### التثبيت والتشغيل

```bash
# الاستنساخ
git clone https://github.com/PinkyBrain-ai/pinkybrain.git
cd PinkyBrain

# التشغيل
python3 src/pinkybrain_v5.py

# أو مع ملف إعدادات
python3 src/pinkybrain_v5.py --config config/bug.json
```

### ربط شبكتك

```json
{
  "node_name": "my-node",
  "private": {
    "p2p_secret": "your-shared-secret-here",
    "peers": [
      {"name": "other-node", "host": "192.168.1.100", "port": 8080}
    ]
  },
  "public_mesh": {
    "enabled": false
  }
}
```

هذا كل شيء. شبكتك الخاصة تعمل فوراً. هل تريد الانضمام إلى الشبكة العامة؟ اضبط `"enabled": true` واختر ما تريد مشاركته.

## 🔑 الميزات الرئيسية

### 🤖 مزودو LLM المتعددون
- **Ollama** (محلي) — متوافق افتراضياً
- **OpenAI** — GPT-4o, GPT-4o-mini, إلخ.
- **Anthropic** — نماذج Claude
- **متوافق مع OpenAI** — LM Studio, vLLM, أي API مخصص
- نماذج من جميع المزودين مشتركة عبر P2P والشبكة العامة

### 🔌 اتصال WebSocket في الوقت الحقيقي
- WebSocket ثنائي الاتجاه على نقطة النهاية `/ws`
- رسائل محددة الأنواع: `query`, `memory_sync`, `memory_update`, `ping/pong`, `auth`
- إعادة اتصال تلقائية مع تراجع أسي

### 🔐 المصادقة اللامركزية
- **هوية Ed25519** — كل عقدة تولد زوج مفاتيحها الخاص
- **سر مشترك HMAC** — بديل أبسط للشبكات الخاصة
- **شبكة ثقة** — العقد تضمن لبعضها البعض، ثقة انتقالية
- **تحديد المعدل** لكل عقدة (خوارزمية دلو الرموز)
- **وضع التخفي** — عقدة مخفية، أقران موثوقون فقط

### 🧠 الذاكرة الموزعة (CRDT)
- **أنواع بيانات مكررة خالية من التعارض** — لا تعارضات دمج أبداً
- **بروتوكول الشائعات** — التغييرات تنتشر تلقائياً
- **ساعات متجهية** — ترتيب سببي للأحداث
- **دعم TTL** — تنتهي صلاحية المدخلات تلقائياً

### 🤖 توجيه نماذج الذكاء الاصطناعي
- **النماذج المحلية أولاً** — الاستعلامات تذهب إلى Ollama المحلي عند الإمكان
- **النماذج السحابية عند الطلب** — بناء الجملة `model:cloud`
- **التحول إلى القرين** — التوجيه إلى قرين إذا النموذج المحلي مشغول
- **إجماع المجموعة** — استعلام نماذج متعددة، إرجاع أفضل إجابة
- **قواطع الدائرة** — إيقاف ضغط الأقران المتعطلين

## 🌐 الشبكة العامة

### بنية الشبكة المزدوجة

```
┌─────────────────────────────────────────────────────┐
│                    العقدة (أنت)                       │
│                                                     │
│  ┌─────────────┐          ┌──────────────────┐     │
│  │ الشبكة الخاصة│          │   الشبكة العامة    │     │
│  │ p2p_secret  │          │   tracker          │     │
│  │ ┌─────────┐ │          │ ┌──────────────┐ │     │
│  │ │ Bug     │◄┼──P2P────┼─┤ العقدة #42    │ │     │
│  │ └─────────┘ │          │ │ 2GB RAM       │ │     │
│  │ ┌─────────┐ │          │ │ 30% CPU       │ │     │
│  │ │ Pinky   │◄┼──P2P────┼─┤ Ollama محلي  │ │     │
│  │ └─────────┘ │          │ └──────────────┘ │     │
│  └─────────────┘          │ ┌──────────────┐ │     │
│                           │ │ العقدة #789   │ │     │
│  ┌─────────────────┐      │ │ 8GB RAM       │ │     │
│  │ حارس الموارد     │      │ │ RTX 4090      │ │     │
│  │ max_ram: 2GB    │      │ │ 4 نماذج       │ │     │
│  │ max_cpu: 30%    │      │ └──────────────┘ │     │
│  │ gpu_share: معطل │      │                  │     │
│  │ priority: محلي   │      │  Tracker:        │     │
│  └─────────────────┘      │  إعلان/قدرات     │     │
│                           └──────────────────┘     │
└─────────────────────────────────────────────────────┘
```

شبكتك **الخاصة** (p2p_secret) معزولة تماماً عن الشبكة **العامة** (Ed25519 + شبكة ثقة). منافذ منفصلة، مصادقة منفصلة، صفر تسريب للبيانات.

### حصص قائمة على المساهمة

| المساهمة | النقاط | الحصة العامة |
|---|---|---|
| لا شيء مشترك | 0 | 1 استعلام / 5 دقائق |
| نموذج واحد مشترك | +20 | 5 استعلامات / دقيقة |
| نموذجان+ مشتركان | +30 | 20 استعلام / دقيقة |
| 2GB RAM مشتركة | +20 | +10 استعلامات / دقيقة |
| GPU مشترك | +20 | +20 استعلام / دقيقة |
| 24 ساعة تشغيل | +10 | +5 استعلامات / دقيقة |

**شارك أكثر = وصول أكثر.** لكن حتى بدون مشاركة شيء، تحصل على استعلام واحد كل 5 دقائق. لا أحد يُحظر.

## 🛡️ حارس الموارد

جهازك يأتي أولاً. حارس الموارد يراقب المعالجة/الذاكرة ويوقف المشاركة العامة تلقائياً عندما تكون مشغولاً.

```python
class ResourceGuard:
    def can_accept_request(self) -> bool:
        cpu_usage = psutil.cpu_percent(interval=0.1)
        ram_usage = psutil.virtual_memory().percent
        
        if self.priority == "local_first":
            if cpu_usage > 70 or ram_usage > 85:
                return False  # المستخدم مشغول
        
        if cpu_usage > self.max_cpu + 40:
            return False
        
        return True
```

**الأولوية المحلية تفوز دائماً.** إذا كان جهازك مشغولاً، يرفض الطلبات العامة. بلا استثناء.

## 🧠 الجدولة التكيفية

الشبكة تقرر أفضل استراتيجية بناءً على عدد الأقران المتاحين. بدون أرقام إصدارات، بدون أوضاع يدوية.

| الأقراء المتاحون | الاستراتيجية | القدرة |
|---|---|---|
| 1–3 | توجيه بسيط | نماذج كاملة على جهاز واحد |
| 4–10 | تقسيم جزئي | نماذج مقسمة إلى 2–4 أجزاء |
| 11–50 | تقسيم كامل + نسخ مزدوج | خط أنابيب متوازي، تكرار |
| 50+ | RAID RAM موزع | قرص افتراضي في RAM، نسخ ثلاثية، جلب مسبق |

**الانتقالات تحدث تلقائياً وبدون انقطاع.** قرين ينضم → المجدول يعيد التوزيع. قرين يغادر → النسخ تتولى. لن تلاحظ شيئاً.

## 💬 مخزن المحادثات الدائم

محادثاتك تبقى على جهازك. نقطة.

- **حفظ تلقائي** — كل رسالة محفوظة محلياً. بدون زر "حفظ".
- **استئناف** — افتح PinkyBrain غداً، محادثاتك موجودة.
- **بحث** — اعثر على أي محادثة سابقة بالكلمة المفتاحية أو التاريخ أو النموذج أو الوسم.
- **تصدير** — Markdown, JSON, نص عادي. بياناتك، تنسيقك.
- **الخصوصية** — المحادثات لا تغادر جهازك أبداً ما لم تزامنها عبر P2P الخاص.
- **التشفير** — تشفير محلي اختياري. حتى الوصول للقرص لا يمكن قراءتها.
- **بدون تتبع** — بدون تحليلات، بدون تدريب على بياناتك.

### مستويات الخصوصية

| المستوى | ماذا يحدث | حالة الاستخدام |
|---|---|---|
| **خاص** (افتراضي) | يبقى محلياً، لا يتم مزامنته أبداً | شخصي، حساس |
| **مزامن** | مزامنة عبر P2P الخاص فقط | بين أجهزتك |
| **مشترك** | مشاركة مع أقران محددين | تعاون |
| **عام** | مساهمة اختيارية في قاعدة معرفة الشبكة | معرفة مجتمعية |

**الافتراضي هو الخاص. دائماً.**

## 🔒 تشفير E2E

عند الاستعلام من الشبكة العامة، بياناتك مشفرة من الطرف إلى الطرف:

1. سؤالك مشفر بمفتاح جلسة
2. كل قرين في خط الأنابيب يفك تشفير جزئه فقط، يحسب، يعيد التشفير
3. أنت فقط يمكنه فك تشفير الإجابة النهائية

**ما يمكن لكل قرين رؤيته:**
| البيانات | مرئي؟ | السبب |
|---|---|---|
| سؤالك الأصلي | ❌ لا | مشفر بمفتاح جلستك |
| الإجابة النهائية | ❌ لا | مشفرة بمفتاح جلستك |
| موترات الإدخال/الإخراج لجزئه | ✅ نعم | ضرورية للحساب |
| بيانات الأجزاء الأخرى | ❌ لا | مشفرة بمفاتيح الأقراء الآخرين |

**هذا ليس وعداً. إنها تشفير.** حتى لو تم اختراق جميع أقراء الشبكة، لا يمكنهم قراءة بياناتك بدون مفتاح جلستك — الذي يوجد فقط على جهازك، فقط أثناء الطلب.

## 📂 shared_models/ — الحد الفاصل بين الخاص والعام

مجلد مخصص هو **الواجهة الوحيدة** بين نماذجك والشبكة العامة.

```
~/.pinkybrain/
├── conversations/        → 🔒 خاص (لا يُشارك أبداً)
├── memory/               → 🔒 خاص (لا يُشارك أبداً)
├── config/               → 🔒 خاص (لا يُشارك أبداً)
├── shared_models/        → 🌐 منطقة المشاركة (مرئية للشبكة)
│   ├── glm-5.1/          → رابط رمزي إلى ~/.ollama/models/glm-5.1
│   ├── llama3/           → نسخة أو رابط رمزي
│   └── mistral/          → نسخة أو رابط رمزي
└── ollama/               → 🔒 تخزين Ollama الخاص
```

```bash
pinkybrain share glm-5.1    # مشاركة نموذج (ينشئ رابط رمزي)
pinkybrain unshare glm-5.1  # إيقاف المشاركة (يزيل الرابط الرمزي فقط)
pinkybrain shared            # قائمة النماذج المشتركة
```

**الشبكة العامة لا تقرأ أبداً خارج `shared_models/`.** إيقاف المشاركة فوري — الشبكة تفقد الوصول فور إزالة الرابط الرمزي.

## 🖥️ واجهة سطح المكتب

4 علامات تبويب، صفر حاجة للطرفية:

- **💬 محادثة** — استعلام نماذج الذكاء الاصطناعي، تاريخ المحادثات، بحث، تصدير
- **📊 مشاركة** — أشرطة المعالجة/الذاكرة/وحدة معالجة الرسوميات، مفاتيح تبديل مشاركة النماذج، إحصائيات المساهمة
- **🔒 الشبكة** — الأقراء الخاصون، عقد الشبكة العامة، التحقق من العزل
- **⚙️ الإعدادات** — اسم العقدة، إعدادات الشبكة العامة، التخزين، عتبات الإيقاف التلقائي

تعمل في أي متصفح على `localhost:8080`. قابلة للتثبيت كتطبيق PWA لسطح المكتب/الهاتف.

## 🔒 الأمان والخصوصية

- **الشبكة الخاصة:** مشفرة بـ p2p_secret (بدون تغيير من v4)
- **الشبكة العامة:** هوية Ed25519 + TLS للنقل
- **تشفير E2E:** الاستعلامات مشفرة من الطرف إلى الطرف أثناء الاستدلال الموزع
- **صفر تسريب للبيانات** بين الشبكة الخاصة والعامة
- **طلبات العامة في صندوق حماية:** بدون وصول إلى الذاكرة الخاصة
- **حارس الموارد:** إيقاف تلقائي للمشاركة عندما يكون حاسوبك مشغولاً
- **وضع التخفي:** شارك المعالجة لكن ابقَ مخفياً على المتتبع
- **صفر تسجيل:** عقد الشبكة العامة لا تخزن الاستعلامات أو الردود أبداً

## 🔄 الانتقال (v4 → v5)

1. v5 **متوافق مع الإصدارات السابقة** مع v4
2. إعدادات الشبكة الخاصة تعمل تماماً كما كانت
3. قسم `public_mesh` **اختياري** — معطل افتراضياً
4. عقد v4 الموجودة يمكنها التواصل مع عقد v5 على الشبكة الخاصة
5. الشبكة العامة **اختيارية:** اضبط `public_mesh.enabled = true`
